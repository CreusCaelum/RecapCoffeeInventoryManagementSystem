# RecapCoffeeInventoryManagementSystem
Inventory Management System

The Recap Coffee Inventory Management System is a web-based application developed to improve the efficiency and accuracy of inventory management in a coffee shop environment. The system replaces the traditional manual inventory process by providing automated stock monitoring, product management, supplier records, and inventory report generation.

The system includes role-based access control, allowing both the owner and staff to perform specific tasks within the system. Staff members can manage inventory, update stock levels, and record ingredient usage through recipes and Bill of Materials (BoM), while the owner can monitor reports and oversee system activities.
